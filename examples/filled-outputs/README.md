# Filled Output Examples

가상의 "재구매 할인 쿠폰" 기능을 기준으로 실제 채워진 산출물 예제다.
템플릿을 복사한 뒤 어떤 수준으로 채워야 하는지 참고용으로 사용한다.

## 가상 시나리오
- 프로젝트: e-commerce 백엔드
- 요청: "재구매 고객을 대상으로 특정 기간 동안 할인 혜택을 주고 재방문을 유도하고 싶다"
- 분류: raw-request
- 프로젝트 유형: brownfield

## 포함 파일
- `status.md` — 기능 상태 추적
- `requirements.md` — 요구사항 정리
- `requirement-verification-questions.md` — 미결 질문
- `unit-of-work.md` — 작업 단위 분해
